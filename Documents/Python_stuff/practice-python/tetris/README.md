# Sand Connect

A colorful Pygame puzzle game where you connect sand particles of the same color.

## Features

- Colorful graphics and animations
- Sound effects and background music
- Pause and resume functionality
- Pixel art style UI

## Installation

### Prerequisites

- Python 3.x
- Pygame
